# Seeded Servicer Profile Picture Prompts (96)

Each business gets ONE of six profile-picture types so the marketplace looks like real, independent companies:

| Type | Meaning |
|------|---------|
| Icon | icon-only logo mark, no text |
| Combo | icon + company name lockup |
| Wordmark | stylized company name typography only |
| Monogram | initials-based lettermark |
| Badge | circular/shield emblem, name integrated in badge |
| Photo | realistic photographic brand image (tools, workspace, food, storefront - no readable faces) |

Types rotate so the 3 merchants in the same category never share a type. Colors and art styles vary per brand - no two should feel like the same designer.

All images: square 1:1, 512×512, centered composition, clean professional finish, suitable as marketplace profile avatar.

Name output file: `backend/uploads/profiles/demo/{Ref}_{ShortName}.png`
